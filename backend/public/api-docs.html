Syntheo API Documentation

Complete REST API documentatie voor het Syntheo zorgsysteem

Base URL: http://localhost:8000/api

Response Format

Alle API responses volgen dit standaard formaat:

{
  "success": true|false,
  "message": "Optional message",
  "data": {} | []
}

Authentication

POST /login

Authenticeer een gebruiker en verkrijg gebruikersgegevens.

Request Body

email string required
Email adres van de gebruiker
password string required
Wachtwoord van de gebruiker

Response

{
  "success": true,
  "data": {
    "user_id": 1,
    "name": "Jan Jansen",
    "email": "jan@example.com",
    "role": "Verpleegster"
  }
}

POST /change-password

Wijzig het wachtwoord van een gebruiker.

Request Body

user_id integer required
current_password string required
new_password string required
new_password_confirmation string required

Belangrijke Opmerkingen

  • Alle timestamps zijn in UTC formaat: YYYY-MM-DDTHH:MM:SS.000000Z
  • Alle datums gebruiken het formaat: YYYY-MM-DD
  • De API handelt automatisch Nederlands ↔ Engels mapping voor urgency en category velden in notes
  • Boolean waarden worden teruggegeven als true of false (niet 1/0)
  • Foreign key constraints worden afgedwongen - het verwijderen van een parent record kan child records cascade deleten
